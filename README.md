RPiHY28bShield
==============
Raspberrypi 2.8" TFT-LCD Shield : HY28a , HY28b Auto install Script

pi@openmake ~ $ git clone https://github.com/rasplay/RPiHY28bShield

pi@openmake ~ $ cd RPiHY28bShield

[example]
pi@openmake ~/RPiHY28bShield $ sh setup.sh hy28b
